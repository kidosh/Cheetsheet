HTML Form Reference

Form Tags

Tag Name Description
<form> Form The Form element is used to collect and send information to an external source.
<input> Input The Input element is used to create interactive controls
<textarea> Textarea The Textarea element is used when creating a text-box for multi-line input
<label> Label The Label element tag defines a label for many form elements.
<select> Select The Select element can be used to create a dropdown list.

Form Attributes

Attribute Name Description
type Type This input elements can be of type of text or number
name Name This input elements is the name of the form
value Value The value of the element is the content in between
for For The attribute of the tag should be equal to the id attribute
action Action The attribute defines what should happen to data when a form is submitted on a web page.
method Method The attribute specifies how to send form-data
id Id The attribute specifies a unique id for an HTML element
min Min The attribute defines the minimum value that is acceptable and valid for the input
max Max The attribute defines the minimum value that is acceptable and valid for the input
maxlength Maxlength The attribute defines the maximum number of characters
minlength Minlength The attribute defines the minimum number of characters
pattern Pattern This is an attribute of the text, tel, email, url, password, and search input types.

Type attributes

Attribute Name Description
text Text Displays a single-line text input field
radio Radio Displays a radio button (for selecting one of many choices)
checkbox Chekbox Displays a checkbox (for selecting zero or more of many choices)
submit Submit Displays a submit button (for submitting the form)
button Button Displays a clickable button
